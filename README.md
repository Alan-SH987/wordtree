# 英语词汇 · 归纳学习法

一份网页互动学习包，改编自 **《怎样学习英语词汇》**（董石祥 编，南宁师专英语科，1978 年油印本）。
保留原书"按音、形、义、根成组归纳"的核心学习方法，将其中的构词法与各类归纳词表，
重新整理成可浏览、可抽认、可测验的现代学习资料。

## 怎么用

在线访问：**https://wordtree.vercel.app**

或直接用浏览器打开 **`index.html`** 即可（无需联网、无需安装）。

也可以起一个本地静态服务器（部分浏览器对 `file://` 限制较多时更稳妥）：

```bash
cd "Eng_Volcabury"
python3 -m http.server 8042
# 然后浏览器访问 http://localhost:8042
```

## 包含 9 个模块

除「其他构词法」是讲解型（仅浏览）外，每个模块都同时支持 **浏览 / 抽认卡 / 测验** 三种方式。

| 模块 | 学习方式 | 内容 |
|------|----------|------|
| 前缀 | 浏览 / 抽认卡 / 测验 | 30 组前缀，按"否定 / 反向 / 重复 / 使役 / 时序 / 方位 / 数量"归类 |
| 后缀 | 浏览 / 抽认卡 / 测验 | 34 组后缀，按"名词 / 形容词 / 动词 / 副词"归类 |
| 其他构词法 | 浏览 | 合成法、转换法、缩略法、屈折法 |
| 同族词 | 浏览 / 抽认卡 / 测验 | 40 个词根家族 |
| 同音异义 | 浏览 / 抽认卡 / 测验 | 57 组音同形异词（sea/see…） |
| 一词多义 | 浏览 / 抽认卡 / 测验 | 43 个一词多义词 |
| 同形异音 | 浏览 / 抽认卡 / 测验 | 23 个重音/读音变化词（record…） |
| 近义词辨析 | 浏览 / 抽认卡 / 测验 | 38 组近义词及用法区别 |
| 反义词 | 浏览 / 抽认卡 / 测验 | 52 对反义词 |

- **点击朗读**：每个英文词旁有小喇叭，点一下用浏览器内置语音（Web Speech API）朗读，离线可用，优先英式发音。
- **抽认卡**：每组随机抽 20 张，优先未掌握的词；翻面后自评"认识 / 还不熟"，进度自动保存。
- **测验**：四选一（同形异音为读法二选一），即时判分。
- **搜索**：右上角可按英文或中文跨模块检索。
- 学习进度保存在浏览器本地（localStorage），换设备不同步。

## 对原书做了哪些"现代化"

原书方法框架完整保留，但 1978 年的内容做了如下处理：

1. **剔除时代政治例词**——如 anti-Japanese War、pre-liberation、warmonger、comradeship，
   以及结束语中的语录引文，替换为中性常用词。
2. **修正扫描/排印错字**——visiter→visitor、Maxist→Marxist、physist→physicist、
   theoritic→theoretic、domitory→dormitory、livehood→livelihood、unify(重复)→unity 等。
3. **补充现代常用词**——smartphone、software、email、internet、recycle、CEO、DIY 等，
   让构词法的例子贴近当代英语。
4. **统一为现代音标**（同音、同形异音模块）。

## 文件结构

```
index.html    页面框架
styles.css    样式（含深色模式、移动端自适应）
data.js       全部词汇数据（window.VOCAB）
app.js        路由、抽认卡引擎、测验引擎、搜索
```

数据与逻辑分离：要增删词条，只改 `data.js` 即可，无需动其他文件。
